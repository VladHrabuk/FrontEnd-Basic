Команди Git

Першочергові налаштування

git config --global user.name "[ім'я]"
Встановлює ім'я, яке буде відображатися в полі автора у виконуваних вами комітів
git config --global user.email "[адреса електронної пошти]"
Встановлює адресу електронної пошти, яка буде відображатися в інформації у виконуваних вами комітів

Створення репозиторію

git init [назва проекту]
Створює новий локальний репозиторій із заданим іменем
git clone [url-адреса]
Скачує репозиторій із всією історією змін

Внесення змін

git status
Перечислює всі нові або змінені файли, які потребують фіксації
git add [файл]
Індексує вказаний файл для коміту
git commit -m "[повідомлення з описом]"
Фіксує проіндексовані зміни і зберігає їх в історію версій

Колективна робота

git branch
Показує список віток коміту
git branch [ім'я вітки]
Створює нову вітку
git switch -c [ім'я вітки]
Переключається на нову і вітку і обновляє робочу директорію до її стану
git branch -d [ім'я вітки]
Видаляє вибрану вітку
