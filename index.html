Селектори

Селектори представляють структуру веб-сторінки. За їх допомогою складають правила для форматування елементів веб-сторінки. Селекторами можуть бути елементи, їх класи та ідентифікатори, псевдокласи та псевдоелементи.

Види селекторів

  • a - любий елемент HTML для якого визначені правила форматування, такі як колір, фон, розмір...
  • #id - визначає унікальне ім'я елемента, для якого задається стиль
  • .class - застосовують, коли потрібно застосувати стиль до одного або кількох елементів веб-сторінки
  • * - універсальний селектор, застосовується до всі елементів веб-сторінки
  • ul li {} - селектор нащадка, застосовуються до елементів всередині елемента-контейнера
  • p > strong - застосовується до всіх дочірніх елементів відносно батьківського
  • h1 + p - сестринські селектори вибирають елементи з групи елементів одного рівня
  • [атрибут] - селектор атрибуту вибирає елементи на основі його ім'я або значення

Завдання №2

Спадкування та каскад — два фундаментальні поняття в CSS, які тісно пов'язані між собою. Спадкування полягає в тому, що елементи успадковують властивості свого батька (елемента, що їх містить). Каскад проявляється у тому, як різні види таблиць стилів застосовуються до документа, та як конфліктуючі правила перевизначають один одного.

Спадкування є механізмом, за допомогою якого певні властивості передаються від предка до його нащадків. Специфікацією CSS передбачено успадкування властивостей, що стосуються текстового вмісту сторінки, таких як color, font, letter-spacing, line-height, list-style. У багатьох випадках це зручно, оскільки не потрібно задавати розмір шрифту та сімейство шрифтів для кожного елемента веб-сторінки.

Каскадування - це механізм, який керує кінцевим результатом у ситуації, коли до одного елемента застосовуються різні CSS-правила. Існує три критерії, які визначають порядок застосування властивостей - правило !important, специфічність та порядок, в якому підключені таблиці стилів.

Для кожного правила браузер обчислює специфічність селектора, і якщо елемент має конфліктні оголошення властивостей, до уваги береться правило, що має найбільшу специфічність. Значення специфічності складається із чотирьох частин: 0, 0, 0, 0.
